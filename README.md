## 技术选型
* 核心框架 spring boot (2.5.4)  
* 持久层 mybatis-plus(3.4.1)
* 数据库连接池 Alibaba Druid
* 前端组件
  > thymeleaf  
  > bootstrap(5.1.1)  
  > jQuery
* 数据库
  > mysql(5.7)
## 启动说明
* 启动前 请配置 application.yml 中相关地址
* 数据库脚本位于 sql 下面，启动前请自行导入
* 配置完成，运行Application
